# 📱 Cyberyan VC/DID Task

## 🛠 Tech Stack

- Expo (Managed Workflow)
- React Native
- TypeScript
- Node.js
- Express

---

## 🚀 Installation & Run

### 1. Clone Repository

```bash
git clone https://github.com/AbolfazlAdhami/cyberyan-task.git
cd cyberyan-task/mobile-app
```

## 2. Back-End

```bash
cd backend
npm install
npm run dev
```

### 3. Mobile App
```bash
cd mobile-app
npm i 
npm start

```


---

## 📌 App Flow

1. Register user (name, email, upload images)
2. Backend returns DID and Verifiable Credential (VC)
3. Wallet screen displays DID and VC fields
4. Audit screen shows SHA-256 issuance hash
